### Bài 3: kiểu dũ liệu trong python
- Python là một ngôn ngữ theo kiểu thông dịch (là ko cần biên dịch), là tự động hiểu, đoán đc kiểu dữ liệu
- Cách kiểm tra kiểu dữ liệu dùng: type()
- Các kiểu dữ liệu cơ bản: int, float, bool, str, complex, NoneType
-     x = 10 (int)
-     y = 1.5 (float)
-     z = True (bool)
-     a = "Hello" (str)
-     b = 1 + 2j (complex)
-     c =  None (NoneType === Null)
